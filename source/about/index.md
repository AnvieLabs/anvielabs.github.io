# Introduction

Hi! I'm **Siddharth Mishra**.  
At present, I'm the only developer working in the Anvie CoreTeam.  Work on Anvie started way back in Feb-Mar of 2023. I started working on a tool that'll help me in my daily work but now it seems it'll be a better idea to convert this to a startup.

# How It Started?

It was supposed to be a GUI tool so I started with building the frontend first instead of backend, because I like to see the backend working with my eyes rather than printing everything on screen. I mean not everything's possible on terminal easily. I wanted to draw trees and graphs (nodes and edges) and other graphics like displaying images etc... that's not possible in terminals easily. You can use something like `ncurses` but I don't want that. If I'm creating something I want it to be easily usable. That's what **Engineering** is. Doing everything quickly just to deliver something is **Business**. I don't want that!

That's when I started working on my own GUI library after trying some already present libraries and experiencing their limitations in one way or other.
